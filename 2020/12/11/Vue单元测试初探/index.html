     由于之前对于单元测试的看法是写起来麻烦而且用处不大，导致自己也没怎么去学习和了解这方面的知识。于是最近试着尝试和使用了一下，下面就简单介绍下vue中如何进行单元测试：


什么是单元测试


❝

单元测试允许你将独立单元的代码进行隔离测试，其目的是为开发者提供对代码的信心。通过编写细致且有意义的测试，你能够有信心在构建新特性或重构已有代码的同时，保持应用的功能和稳定。


❞

测试环境搭建



  • 这里我们使用的是vue3.x
  • 测试框架使用的是jest

❝

1.首先创建一个vue项目
2.在创建的过程中勾选Unit Test 或者 使用vue命令安装


vue add unit-jest

环境搭建好之后可以看到根目录新增一个test文件夹（其中包含了一个helloworld测试demo）和jest.config.js配置文件


❞

测试功能


  • 数据类型

测试组件


  • 具体代码如下

<template>
  <div class=”test_demo”>
    <input v-model=”state.user” />
    <input v-model=”state.password” />
    
    <button @click=”handleSubmit”>submit</button>
    <button @click=”goBack”>back</button>
  </div>
</template>
<script>
import { reactive } from ’vue’;
import { useRouter } from ’vue-router’;
export default {
  components: {},
  props: {
    msg: String,
  },
  setup() {
    const state = reactive({
      user: ’’,
      password: ’’,
      arr: [],
    });
    const router = useRouter();
    function handleSubmit() {
      if (!state.user || !state.password) {
        console.log(‘不能为空’);
        return false;
      }
      return true;
    }
    function goBack() {
      router.back();
    }
    return {
      state,
      handleSubmit,
      goBack,
    };
  },
};
</script>

测试用例


  • 在tests\unit下创建test.spec.js(名字可以随意命名 但要是xx.spec.js or xx.spec.ts)

数据类型测试

import TestDemo from ‘@/components/testDemo.vue’;
describe(‘组件变量类型’, () => {
  it(‘数组类型测试’, () => {
    expect(Array.isArray(TestDemo.setup().state.arr)).toBeTruthy();
  });
  it(‘字符串类型测试 ’, () => {
    expect(typeof TestDemo.setup().state.user).toBe(‘string’);
  });
});

函数返回测试

describe(‘组件函数返回’, () => {
  it(‘submit 函数’, () => {
    expect(TestDemo.setup().handleSubmit()).toBeFalsy();
  });
});

prop测试

describe(‘prop 验证’, () => {
  it(‘props 测试’, () => {
    const msg = ‘hello world’;
    const wrapper = shallowMount(TestDemo, {
      props: { msg },
    });
    expect(wrapper.vm.msg).toMatch(msg);
  });
});

接口测试

describe(‘检查接口’, () => {
  it(‘检查接口’, async () => {
    let { data } = await Axios.get(‘http://192.168.1.145:3000/submit');
    expect(data.code).toBe(200);
  });
});

完整代码

import TestDemo from ‘@/components/testDemo.vue’;
import { shallowMount } from ‘@vue/test-utils’;
import Axios from ‘axios’;

    expect(typeof TestDemo.setup().state.user).toBe(‘string’);
  });
});
describe(‘组件函数返回’, () => {
  it(‘submit 函数’, () => {
