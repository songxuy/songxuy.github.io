前端水印生成

十二月 01, 2020
  • JavaScript
  • 水印

     对于一些技术支持型的网站，水印是一种很好的保护知识产权的方式。前几天看了一篇水印介绍的文章再结合自己之前的简单使用，以下就介绍几种简单的生成水印的方法：


水印的介绍



  • 水印一般分为明水印和暗水印

❝

明水印:
主要就是我们通常看到的那一种，在页面上我们能看到相应的logo、name等。生成方式主要可以归为两类，一种是纯html元素（纯div），另一种则是通过生成背景图片（canvas/svg）


❞

❝

暗水印:
暗水印是一种肉眼不可见的水印方式，可以保持图片美观的同事，保护资源的版权。生成方式也比较多，常见的是通过修改RGB分量值的小量变动、DWT、DCT和FFT等方法。


❞


  • 很明显暗水印的保护效果会比明水印好很多，无论从实现的效果还是防御能力上看。这篇文章我们主要来介绍一下明水印的几种简单的实现。

明水印的实现



  • 从我们的视觉上能感觉到水印和我们的内容类似于两个图层，相互叠加之后达到了最终的效果。下面就从背景和div来分别实现水印。

Canvas实现

❝

canvas的实现很简单，主要是利用canvas 绘制一个水印，然后将它转化为 base64 的图片，然后将获取的 url 填充在一个元素的背景中，然后我们设置背景图片的属性为重复。


❞


  • css

.watermark {
  position: fixed;
  top: 0px;
  right: 0px;
  bottom: 0px;
  left: 0px;
  pointer-events: none;
  background-repeat: repeat;
}


  • JS

const createWaterMark = () => { 
  const angle = -20;
  const txt = ‘秋风的笔记’
  const canvas = document.createElement(‘canvas’);
  canvas.width = 180;
  canvas.height = 100;
  const ctx = canvas.getContext(‘2d’);
  ctx.clearRect(0, 0, 180, 100);
  ctx.fillStyle = ‘#000’;
  ctx.globalAlpha = 0.1;
  ctx.font = 16px&nbsp;serif
  ctx.rotate(Math.PI / 180  angle);
  ctx.fillText(txt, 0, 50);
  return canvas.toDataURL();
}
const createCanvas = () => {
