Vue自定义指令

九月 16, 2023
  • 自定义指令
❝

对于Vue的自定义指令在我们工作中可能用到的不是很多，但是在某些特定场景下使用自定义指令来实现的话能够提升效率（例如按钮级的权限控制、loading等）
下面就介绍下自定义指令来实现权限控制。




  • v-permission

import { createApp } from ‘vue’
import { createPinia } from ‘pinia’
import App from ‘./App.vue’
import router from ‘./router’
const app = createApp(App)
app.directive(“permission”, (el, binding) => {
  //这会在 mounted 和 updated 时都调用
  const roleList = [‘admin’, ‘user’]
  const flag = binding.value || “”
  if (roleList.includes(flag)) {
    el.style.display = “”
  } else {
    el.parentNode && el.parentNode.removeChild(el)
  }
})
app.use(createPinia())
app.use(router)
app.mount(‘#app’)


  • 指令使用

// 由于我们是全局注册的指令，可以在组件中直接使用
<button @click=“getChild” v-permission=“‘admin’”>授权</button>
