border动画demo

一月 27, 2021
  • Css

     相信大家都写过不少关于border的样式，但是加上动画的话可能就会少一点。之前看了一篇文章，里面关于border动画的介绍都挺不错的，自己也简单的总结下：


边框长度的变化


❝

效果：hover的时候外边框再变化闭合，主要是通过两个伪元素分别来控制左上和右下两边边框的变化


❞



  • 具体代码

<!DOCTYPE html>
<html lang=“en”>
<head>
  <meta charset=“UTF-8”>
  <meta name=“viewport” content=“width=device-width, initial-scale=1.0”>
  <title>border1</title>
</head>
<style>
  div {
  position: relative;
  width: 200px;
  height: 50px;
  border: 1px solid #03A9F3;
  margin: 30px;
}
div::before, div::after {
  content: “”;
  position: absolute;
  width: 20px;
  height: 20px;
}
div::before {
  top: -5px;
  left: -5px;
  border-top: 1px solid #03A9F3;
  border-left: 1px solid #03A9F3;
  transition: all .2s;
}
div::after {
  right: -5px;
  bottom: -5px;
  border-bottom: 1px solid#03A9F3;
  border-right: 1px solid#03A9F3;
  transition: all .2s;
}
div:hover::before, div:hover::after {
  width: calc(100% + 9px);
  height: calc(100% + 9px);
}
</style>
<body>
  <div></div>
</body>
</html>

虚线边框动画


❝

效果： 类似dashed属性，但是能动起来。这里主要是通过背景渐变和动画来实现，或者使用border+background-position来实现


❞



  • 具体代码

<!DOCTYPE html>
<html lang=“en”>
<head>
  <meta charset=“UTF-8”>
  <meta name=“viewport” content=“width=device-width, initial-scale=1.0”>
  <title>border2</title>
</head>
<style>
  div {
  position: relative;
  width: 200px;
  height: 50px;
  margin: 30px;
  background: 
      linear-gradient(90deg, #03A9F3 50%, transparent 0) repeat-x,
      linear-gradient(90deg, #03A9F3 50%, transparent 0) repeat-x,
      linear-gradient(0deg, #03A9F3 50%, transparent 0) repeat-y,
      linear-gradient(0deg, #03A9F3 50%, transparent 0) repeat-y;
  background-size: 4px 1px, 4px 1px, 1px 4px, 1px 4px;
  background-position: 0 0, 0 100%, 0 0, 100% 0;
}
div:hover {
    animation: borderMove .1s infinite linear;
}
@keyframes borderMove {
    100% {
        background-position: 4px 0, -4px 100%, 0 -4px, 100% 4px;
    }
}
</style>
<body>
  <div></div>
</body>
</html>


边框渐变移动


❝

效果： 边框由几种颜色连接而成，并且转动。主要是通过伪元素+overflow来实现（伪元素宽高可以根据父元素来设置）


❞



  • 具体代码

<!DOCTYPE html>
<html lang=“en”>
<head>
  <meta charset=“UTF-8”>
  <meta name=“viewport” content=“width=device-width, initial-scale=1.0”>
  <title>border3</title>
</head>
<style>
  div {
    position: relative;
    width: 200px;
    height: 150px;
    margin: 30px;
    overflow: hidden;
    border-radius: 8px;
  }
  div::after{
    content: ‘’;
    position: absolute;
    left: -50%;
    top: -50%;
    width: 200%;
    height: 200%;
    background-repeat: no-repeat;
    background-size: 50% 50%, 50% 50%;
    background-position: 0 0, 100% 0, 100% 100%, 0 100%;
    background-image: linear-gradient(#399953, #399953), linear-gradient(#fbb300, #fbb300), linear-gradient(#d53e33, #d53e33), linear-gradient(#377af5, #377af5);
    animation: rotate 4s linear infinite;
  }
  div::before{
    content: ‘’;
    position: absolute;
    width: calc(100% - 10px);
    height: calc(100% - 10px);
    background-color: #fff;
    top: 5px;
    left:5px;
    z-index: 2;
  }
  @keyframes rotate {
    100% {
      transform: rotate(1turn);
    }
  }
</style>
<body>
  <div></div>
</body>
</html>

clip-path的使用


❝

效果： border跟随效果，主要是通过clip-path+动画来实现


❞



  • 具体代码

<!DOCTYPE html>
<html lang=“en”>
<head>
  <meta charset=“UTF-8”>
  <meta name=“viewport” content=“width=`, initial-scale=1.0”>
  <title>border4</title>
</head>
<style>
  div{
    position: relative;
    width: 200px;
    height: 100px;
    margin: 30px;
    border: 2px solid gold;
    border-radius: 10px;
    background: gold;
    transition: all .3s;
    cursor: pointer;
  }
  div::before, div::after {
    content: “”;
    position: absolute;
    top: -10px;
    left: -10px;
    right: -10px;
    bottom: -10px;
    border: 2px solid gold;
    border-radius: 10px;
    animation: clippath 3s infinite linear;
  }
  div::after {
      animation: clippath 3s infinite -1.5s linear;
  }
  @keyframes clippath {
    0%,
    100% {
        clip-path: inset(0 0 95% 0);
    }
    25% {
        clip-path: inset(0 95% 0 0);
    }
    50% {
        clip-path: inset(95% 0 0 0);
    }
    75% {
        clip-path: inset(0 0 0 95%);
    }
}
</style>
<body>
  <div></div>
</body>
</html>

借助overflow


❝

效果： hover的时候边框出现类似于第一个。主要通过overflow:hidden+transfrom来实现


❞



  • 具体代码

<!DOCTYPE html>
<html lang=“en”>
<head>
  <meta charset=“UTF-8”>
  <meta name=“viewport” content=“width=device-width, initial-scale=1.0”>
  <title>border5</title>
</head>
<style>
  div{
    position: relative;
    width: 200px;
    height: 100px;
    margin: 30px;
    overflow: hidden;
  }
  div::after {
    content: “”;
    position: absolute;
    top: 4px;
    bottom: 4px;
    right: 4px;
    left: 4px;
    line-height: 92px;
    font-size: 24px;
    background:  yellowgreen;
    border: 2px solid #fff;
    cursor: pointer;
    color: yellowgreen;
    }
    
    div::before {
        content: “”;
        position: absolute;
        top: 0px;
        bottom: 0px;
        right: -20px;
        left: 0px;
        background: yellowgreen;
        transform: rotateZ(-90deg) translate(-100%, -100%);
        transform-origin: top left;
        transition: transform .3s;
        transition-timing-function: linear;
    }
    
    div:hover::before {
        transform: rotateZ(0deg) translate(0%, -0%);
    }
</style>
<body>
  <div></div>
</body>
</html>
